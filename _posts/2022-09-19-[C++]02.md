비트마스킹 연산 간단하게 정리

### i번째 bit가 1인지 아닌지 체크

```c++
bit & (1<<i)
```

### i번째 bit를 1로 만들기

```c++
bit |= (1<<i)
```

### i번째 bit를 0으로 만들기

```c++
bit &= ~(1<<i)
```

### i번째 bit가 0이면 1로, 1이면 0으로

```c++
bit ^= (1<<i);
```
